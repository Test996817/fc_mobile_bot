# UNIVERSE OF HEROES - Tournament Format

## Описание формата
- 64 игрока, 4 группы по 16
- 6 раундов в групповом этапе
- Playoff: 1/8 → 1/4 → 1/2 → Финал
- Best of 5 (нужно 3 победы)
- Финал: 11 игр, победитель по сумме голов

## Группы
| Ключ | Topic | Группа |
|------|-------|--------|
| A | МСТИТЕЛИ | Группа A |
| B | СТРАЖИ ГАЛАКТИКИ | Группа B |
| C | ЛИГА СПРАВЕДЛИВОСТИ | Группа C |
| D | ОТРЯД САМОУБИЙЦ | Группа D |

## Игроки (64)
См. PLAYERS.md

## Команды Admin
- `/nextround [A/B/C/D/playoff]` - следующий раунд
- `/standings [A/B/C/D]` - таблица группы
- `/elo` - таблица ELO
- `/tp [ник]` - техническое поражение (74:0)
- `/replace [old] [new]` - замена игрока
- `/cancelmatch [ник1] [ник2]` - отмена матча
- `/group [A/B/C/D]` - таблица в topic
- `/initplayers` - инициализация 64 игроков
- `/clearplayers` - очистка игроков

## Команды Player
- `!nick [ник]` - установить ник
- `!matches [ник]` - мои матчи
- `!profile` - профиль
- `!commands` - список команд

## Структура БД
- players - игроки
- elo - рейтинг ELO
- group_matches - матчи групп
- group_standings - таблица групп
- playoff_matches - матчи плей-офф
- admins - админы

## Формат результата
- Скриншот с подписью: @Player1 - @Player2
- OCR распознаёт счёт или ручной ввод

## Плей-офф
- Отдельный topic: ПЛЕЙ-ОФФ РЕЗУЛЬТАТЫ
- 1 сообщение, бот редактирует после каждого матча
- Формат:
  - 1/8, 1/4, 1/2: Player1 3 - 1 Player2 (Best of 5)
  - Финал: Player1 15 - 11 Player7 (сумма голов)

## ELO
- Стартовый рейтинг: 1000
- K-factor: 32
- Сохраняется после каждого матча
